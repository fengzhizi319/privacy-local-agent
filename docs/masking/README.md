## 快速开始

1. 阅读 [prd.md](./prd.md) 了解产品需求。
2. 阅读 [design.md](./design.md) 掌握字段识别与脱敏规则。
3. 查看 [examples.md](./examples.md) 快速上手。
4. 开发时参考 [api_reference.md](./api_reference.md)。
5. 部署与排障参考 [ops.md](./ops.md)。
6. 编写测试参考 [testing.md](./testing.md)。
